# Meetup Town
An online meeting app runs on browser.

## Getting started
### Prerequisites
- `Nodejs`
- `npm`

### How to install
1. `npm install`
2. `npm run dev` 
3. Open [localhost:8000](http://localhost:8000) on your browser
